A Linguagem de Marcação de Hipertexto (HTML) permite que os desenvolvedores desenhem como serão exibidos pelo navegador os elementos da página, como texto, hiperlinks e arquivos multimídia.

Trata-se de uma linguagem de marcação, amplamente utilizada com HTML e representa diversas possibilidades para a formatação. O CSS ajuda a editar, alinhar, remover e trabalhar no espaço entre elementos de uma página.
